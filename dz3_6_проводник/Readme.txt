Написать программу «Проводник»
■ При первом запуске программа отображает список доступных дисков;
■ Программа должна содержать дерево дисков, строку адреса, меню,
 панель инструментов и окно для отображения содержимого папки;
■ Дерево дисков отображает только диски и папки (можно
реализовать с помощью ListBox);
■При двойном щелчке по папке – окно содержимого отображаются 
файлы и подпапки;
■ Программа должна иметь развернутое меню, контекстное
меню и возможность работы с горячими клавишами.
